1.Place the valid JAD Data and Ops Data source files in the same folder as  JeppesenExtractor.jar
2. JAD Data Base should be named JAD.csv
3. Ops Data Base should be named OpsData.csv
4. Run the JeppesenExtractor (Alternate: Run ExtractorRunner.bat)
5. JeppesenExtractor generates new WPTData.csv and WPTDataRaport.txt

Known problems:
If the WPTData doesn't appear after running the JeppesenExtractor download the latest Java JRE from:
http://www.oracle.com/technetwork/java/javase/downloads/jre8-downloads-2133155.html
